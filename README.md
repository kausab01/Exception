# ⚠️ Custom Exception Handling in Java (Age Validation)

## 📌 Project Overview

This project demonstrates **User-Defined Exception Handling in Java**.
It validates a user's age and throws a custom exception if the age is not eligible according to the defined condition.

The program shows how developers can create their own exception classes and use them for real-world validation scenarios.

---

## 🎯 Objective

The objective of this project is to understand:

* How to create a **Custom Exception** in Java
* How to throw exceptions using `throw` keyword
* How to handle validation logic using exception handling
* How exception flow works in real-time programs

---

## 🛠️ Technologies Used

* **Language:** Java
* **Concept:** Exception Handling
* **IDE:** Eclipse / VS Code / IntelliJ
* **Platform:** Console Application

---

## ⚙️ Features

✔ User-defined Exception Class (`Age`)
✔ Age Validation Logic
✔ Throws Exception if Age ≤ 18
✔ Demonstrates `throw` and `throws` usage
✔ Runtime Input using Scanner
✔ Real-world Validation Example


## 📊 Program Behavior

| Age Entered | Output                   |
| ----------- | ------------------------ |
| Above 18    | Valid User               |
| 18 or Below | Custom Exception Message |
